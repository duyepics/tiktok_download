# TikTok Downloader Tool

Một công cụ đơn giản để tải video TikTok không có hình mờ (watermark).

## Tính năng
- Tải video TikTok chất lượng cao.
- Không dính watermark.
- Giao diện hiện đại, dễ sử dụng.

## Công nghệ sử dụng
- **Backend**: Node.js, Express
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)

## Cách cài đặt và chạy
1. Clone dự án:
   ```bash
   git clone https://github.com/duyepics/tiktok_download.git
   ```
2. Cài đặt dependencies:
   ```bash
   npm install
   ```
3. Chạy server:
   ```bash
   node server.js
   ```
4. Truy cập: `http://localhost:3000`
